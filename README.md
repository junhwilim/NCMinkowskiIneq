# NCMinkowskiIneq
MATLAB codes for computations in Lim, Noncommutative Minkowski integral inequality and a unitary categorification criterion for fusion rings, 2026. arXiv:2601.13490.\n
To load computation result, run the command "load('FusionRingDataSet_Solutions.mat', 'FusionRingDataSet_Solutions')".
To run a new computation, run FusionRingDataSet0.m.
